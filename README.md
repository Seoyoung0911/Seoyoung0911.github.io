# Seoyoung Cho Personal Homepage

Static GitHub Pages homepage for Lab 12 / Homework 6. This site is intended for:

```text
https://seoyoung0911.github.io/
```

## Files

- `index.html`: main homepage
- `styles.css`: responsive styling
- `assets/profile.png`: generated avatar placeholder
- `assets/cv.pdf`: downloadable CV generated from the supplied CV content
- `DESIGN.md`: selected design direction

## GitHub Pages

Upload these files to the root of a public repository named:

```text
Seoyoung0911.github.io
```

Then enable GitHub Pages from `Settings > Pages`, using `main / root`.
