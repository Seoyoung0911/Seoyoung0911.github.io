# Design Direction

Theme: calm robotics portfolio

Principles:
- Use a clean academic portfolio structure with strong section anchors.
- Keep the first view personal and direct: name, field, contact, profile image, and CV link.
- Use a balanced palette of ink, teal, white, and warm accent colors.
- Make the page readable on mobile and desktop without external libraries.
- Prioritize GitHub Pages compatibility with static HTML, CSS, and local assets only.

Required sections:
- About
- Resume
- Projects
- Courses
- AI Usage
- Contact
